# snookerGame
A 2D Snooker Game developed using Cocos2d-x, focusing on realistic ball physics, smooth gameplay, and clean game architecture.
This project demonstrates core game development concepts such as collision handling, turn-based gameplay logic, and physics-based movement.
**Features**

  Realistic snooker ball physics

  Smooth cue control & shot mechanics

  Collision detection and ball pocketing

  Optimized performance using Cocos2d-x best practices

  Cross-platform support (Android / iOS / Desktop)
